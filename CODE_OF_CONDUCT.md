# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- Be respectful and inclusive.
- Assume good intent. Give constructive feedback.
- Harassment, personal attacks, and discriminatory behavior are not tolerated.

For concerns, open a confidential issue or email the maintainers.
